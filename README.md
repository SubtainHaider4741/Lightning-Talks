# ggrepel Lightning Talk

# Topic

The topic of this presentation is `ggrepel`, an R package that improves text labels in `ggplot2` graphs. It helps avoid overlapping labels, which makes plots easier to read and present.

## Files in this repo

- `ggrepel_vignette_example.R`  
  A worked example based on the main idea from the `ggrepel` documentation and vignette.

- `ggrepel_new_example.R`  
  My own example using the `mpg` dataset to label selected cars in a scatterplot.
